🔴 PROBLEM: "Failed to generate Zoom link. Please try again or contact support"

📊 ROOT CAUSE: Zoom API credentials are expired or invalid


═══════════════════════════════════════════════════════════════════════
🎯 SOLUTION 1: FIX ZOOM CREDENTIALS (BEST - 5 MINUTES)
═══════════════════════════════════════════════════════════════════════

Quick Steps:
1. Visit: https://marketplace.zoom.us/develop/create
2. Create "Server-to-Server OAuth" app
3. Get: Account ID, Client ID, Client Secret
4. Update .env file with new credentials
5. Restart server: python manage.py runserver
6. Test: python test_zoom.py

📖 Detailed Guide: See ZOOM_CREDENTIALS_FIX.txt


═══════════════════════════════════════════════════════════════════════
🎯 SOLUTION 2: MANUAL ZOOM LINK ENTRY (IMMEDIATE - 2 MINUTES)
═══════════════════════════════════════════════════════════════════════

Use Admin Panel:
──────────────────────────────────────────────────────────────────────
1. Login to admin: http://localhost:8000/admin
   Username: admin
   Password: (your password)

2. Go to "Appointments"

3. Click on the appointment that needs Zoom link

4. Scroll to "Zoom Meeting" section (expand it)

5. Manually enter:
   - Zoom Join URL: https://zoom.us/j/xxxxxxxxxxxxx
   - Zoom Password: (if any)

6. Click "Save"

7. ✅ Patient can now see and use the Zoom link!


How to Get Manual Zoom Link:
──────────────────────────────────────────────────────────────────────
1. Login to Zoom: https://zoom.us/
2. Click "Schedule a Meeting"
3. Set date and time (match appointment time)
4. Click "Save"
5. Copy the meeting link
6. Paste in admin panel
