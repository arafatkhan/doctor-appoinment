💳 WHEN YOU GET bKASH CREDENTIALS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

একবার bKash credentials পেলে:

STEP 1: Get Credentials
──────────────────────────────────────────────────────────────────────
Visit: https://developer.bka.sh/
Register and get:
  - App Key
  - App Secret
  - Username
  - Password


STEP 2: Update .env File
──────────────────────────────────────────────────────────────────────
Open: .env

Update these lines:
BKASH_APP_KEY=your_actual_app_key
BKASH_APP_SECRET=your_actual_app_secret
BKASH_USERNAME=your_actual_username
BKASH_PASSWORD=your_actual_password


STEP 3: Enable Real Payment
──────────────────────────────────────────────────────────────────────
Open: appointments/views.py

In process_bkash_payment() function:
  1. Comment out TEST MODE section
  2. Uncomment REAL bKash INTEGRATION section

In payment_callback() function:
  1. Comment out TEST MODE section
  2. Uncomment REAL bKash INTEGRATION section


STEP 4: Restart Server
──────────────────────────────────────────────────────────────────────
python manage.py runserver

✅ Now REAL bKash payment gateway will work!
