📱 HOW THE SYSTEM WORKS:
═══════════════════════════════════════════════════════════════════════

Login Flow:
──────────────────────────────────────────────────────────────────────
1. User goes to: http://localhost:8000/login
2. Enters username and password
3. System checks:
   - Is user a doctor? → Redirect to doctor dashboard
   - Is user admin? → Redirect to admin panel
   - Is user patient? → Redirect to patient dashboard
4. User sees appropriate interface


Doctor Dashboard:
──────────────────────────────────────────────────────────────────────
1. Doctor logs in
2. System automatically detects doctor profile
3. Loads ONLY that doctor's appointments
4. Shows statistics, today's schedule, patients
5. All actions limited to doctor's own data


Admin Dashboard:
──────────────────────────────────────────────────────────────────────
1. Admin logs in
2. Can access any doctor's dashboard
3. Must select doctor from list first
4. Can view all doctors' appointments
5. Full system control
